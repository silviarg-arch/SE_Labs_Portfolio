\# Lab 4 – UML Behavioral Diagrams



\## Description

In this lab, we created behavioral diagrams to represent system dynamics and processes.



\## ATM System (Sequence Diagram)

This diagram shows the interaction between Customer, ATM, and Bank System during a transaction. It includes message flow and alternative scenarios such as correct or incorrect PIN.



\## Medical Clinic System (State Diagram)

The diagram represents the different states of an appointment, such as scheduled, in progress, and completed. It shows how the system changes state depending on user actions.



\## Car Insurance System (Activity Diagram)

This diagram models the process of handling an insurance claim, including steps such as submission, validation, approval, and payment.



\## Pet Care System (Sequence Diagram)

An additional sequence diagram was created to represent the appointment booking process, showing interactions between the user and the system.
