# Software Engineering Labs Portfolio

This repository contains all the work developed for the Software Engineering course, including UML diagrams and a complete SRS project.

## 📁 Repository Structure

### 🔹 LABS
Contains all laboratory assignments:

- **Lab 2 – Use Case Diagrams**
  - ATM System
  - Medical Clinic System
  - Car Insurance System

- **Lab 3 – Class Diagrams**
  - Object-oriented system modeling with relationships and attributes

- **Lab 4 – Behavioral Diagrams**
  - Sequence Diagram (ATM System)
  - State Diagram (Medical Clinic System)
  - Activity Diagram (Car Insurance System)

### 🔹 SRS – Pet Care System
Includes a complete Software Requirements Specification based on IEEE 830:

- Use Case Diagram
- Class Diagram
- Sequence Diagram
- System description and requirements

## 🛠 Tools Used
- UML modeling tools (Lucidchart / alternatives)
- Git & GitHub for version control
